# npx create-react-app --template typescript
=> 타입스크립트 템플릿으로 리액트 앱 생성

# npm install styled-components
=> 추가적으로 npm install --save-dev @types/styled-components를 통해 타입 선언

# npm install react-router-dom, react-query
=> 라우터 보다 쉽게 구현가능하도록 도와줌

# npm install react-apexcharts
=> 리액트에서 유용하게 사용가능한 차트 제공

# npm install recoil
=> 리액트 상태 관리
